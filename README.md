# Exam Question Bank System

Hệ thống thi trắc nghiệm 100 câu hỏi được phát triển bằng HTML, CSS, JavaScript (Frontend) và Node.js Express + MongoDB (Backend).

## Tính năng
- Thi trắc nghiệm theo bộ đề (100 câu/đề).
- Phản hồi đúng/sai tức thì khi chọn đáp án.
- Tự động lưu bài làm dở (Drafts).
- Bảng xếp hạng (Leaderboard) theo số câu đúng.
- Xem lại lịch sử thi chi tiết.
- Tối ưu hiển thị nội dung XML/WSDL.

## Deploy trên Vercel
Dự án đã được cấu hình để deploy lên Vercel. Hãy nhớ thiết lập các biến môi trường `MONGODB_URI` và `MONGODB_DB`.
